# Swift Notes

A personal Swift playground covering core language fundamentals — built while working through the language from scratch.

---

## Topics Covered

### Basics
- Variables & constants (`var`, `let`)
- Type inference vs. type annotations
- String interpolation and common string methods

### Collections
- **Arrays** — appending, removing, sorting, reversing
- **Dictionaries** — default values, empty init, key overwriting
- **Sets** — deduplication, fast lookup, `insert`
- **Tuples** — returning multiple values from functions

### Control Flow
- `if` / `else if` / `else`
- `switch` statements with enums
- Ternary operator `? :`
- `for-in` loops, ranges (`...` and `..<`)
- `while` loops

### Functions
- Parameters, return types, default values
- Multiple return values via tuples
- Error handling with `throws`, `do/try/catch`

### Closures
- Closure syntax and the `in` keyword
- Passing closures as function parameters
- Trailing closure syntax
- Using `filter`, `sorted`, `map` with closures

### Structs & Classes
- Struct properties and methods
- Classes, inheritance, `override`, `final`
- `super.init`, `deinit`, reference vs. value types
- Deep copies

### Protocols & Extensions
- Defining and conforming to protocols
- Protocol properties (`get` / `get set`)
- Protocol-oriented programming with default implementations
- Extending built-in types (`String`, `Array`, `Collection`, `Numeric`)
- Opaque return types (`some`)

### Optionals
- `if let` unwrapping
- `guard let` with early return
- Nil coalescing `??`
- Optional chaining `?.`
- `try?` with throwing functions
